ודוגמאות:

עבור המערכים הבאים ומספר m=5:

int[] arr1 = { 2, 3, 4, 1 };

int[] arr2 = { 2, 4, 6, 3 , 9};

int[] arr3 = { 2, 3, 6, 5 };

int m = 5;

IsValidArray(arr1, m);       →  true

IsValidArray(arr2, m);       →  false

IsValidArray(arr3, m);       →  false
